## Висновки до завдання 2

Функція для тестування - f(x) = x^3, a = 0, b = 2.
Теоретичне значення має бути (b^4)/4 - (a^4)/4 = 4.

Результат роботи програмного коду:
```
Theoretical value: 4.0
Average value for 100 experiments: 3.998687999999999
Average value for 10000 experiments: 4.00019861333333
```

1. При збільшені кількості експериментів у 100 разів точність підвищидась на 1 десятковий знак.
1. Обчислення 10000 експериментів зайняло кілька хвилин в тестовому середовищі.
